Module str

String Manipulation.

When building up strings use a strbuf, which is a sequence of chars; can add characters and strings, formatted strings (like str_fmt ) and also insert and remove substrings. The actual string is always *S but use strbuf_tostring(S) to properly clean up and dispose of the sequence.

str_split creates a refcounted array of strings, splitting using a delimiter. str_concat works the other way, joining an array of strings with a separator.

Then there are searching operations which return a boolean or integer index.

strbuf_addr (sp, s, i1, i2) append a range of chars from a string.
