pairmap (fun, t, ...) line 484
call the function with the key and value pairs from a table. The function can return a value and a key (note the order!). If both are not nil, then this pair is inserted into the result. If only value is not nil, then it is appended to the result.

Parameters:

  • fun func A function which will be passed each key and value as arguments, plus any extra arguments to pairmap.
  • t tab A table
  • ... optional arguments

Usage:

  • pairmap(function(k,v) return v end,{fred=10,bonzo=20}) is {10,20} _or_ {20,10}
  • pairmap(function(k,v) return {k,v},k end,{one=1,two=2}) is {one={'one',1},two={'two',2}}
