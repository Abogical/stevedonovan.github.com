Module mod

Markdown-flavoured and very simple no-structure style.

Here the idea is to structure the document entirely with Markdown.

Using the default markdown processor can be a little irritating: you are required to give a blank line before starting lists. The default stylesheet is not quite right, either.

lpeg.Cc ([, ...)
Creates a constant capture. This pattern matches the empty string and produces all given values as its captured values.
split (s)
Split a string str. Returns the first part and the second part, so that combine(first,second) is equal to s.
split_table (text)
Split a string text into a table.

Returns:

  • name the name of the text
  • pairs an array of pairs
    • key
    • value
  • length
